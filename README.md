# 💤 A LazyVim Config

Supports web developement. Uses Volar with takeover mode as tsserver replacement.

### To clone

```sh
git clone git@github.com:srav001/LazyVim-config.git ~/.config/nvim
```

### To Remove

```sh
rm -rf ~/.config/nvim && rm -rf mv ~/.local/share/nvim && rm -rf mv ~/.local/state/nvim && rm -rf mv ~/.cache/nvim
```
